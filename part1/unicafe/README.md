# Full Stack Open Part 1-2 Unicafe

This web application collects customer feedback with three options: **good**, **neutral**, **bad**.

The application displays: 
- Total number of feedbacks
- Average score
- Percentage of positive feedback

## Application Setup

Follow those command below:

```bash
# Install dependencies
npm install
# Run the Application
npm run dev
```

Then you can access the app at: http://localhost:5173/
