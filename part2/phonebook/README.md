A simple phonebook application built with React that allows users to manage contacts.

## Features
- Add new contacts with name and phone number
- Filter contacts by name
- Update existing contact numbers
- Delete contacts
- Data persistence using `json-server`
- Success and error notifications

## Technologies
- React
- Axios
- json-server

## Prerequisites
- Node.js (v18 or newer recommended)

## Installation
Install project dependencies
```bash
npm install
```
Start the backend
 ```bash
npm run server
```
Start the frontend development server on new terminal
```bash
npm run dev
```

The application will be available at: http://localhost:5173
